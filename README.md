# ALD Vacuum Simulation (MATLAB)

This project simulates how pressure changes inside an ALD chamber during pulse and purge cycles. It uses a simple mass balance with effective pump speed to show how gas flows in and gets removed over time.

The model also looks at flow regime using Knudsen number, includes conductance limits from the exhaust line, and checks how things like purge time and pump speed affect performance. There’s also a basic particle simulation to visualize how gas gets cleared out.

Overall, it’s a simplified model to understand how vacuum and purge behavior works in an ALD system.
